# AstraMig CLI

A standalone CLI that clones the LibraryManagement project into a local folder.

## Commands

- `astramig init` clones into `./AstraMig`
- `astramig init <dir>` clones into `./<dir>`

## Development

```bash
npm install
npm run build
node dist/index.js init
```

## Standalone binaries

```bash
npm run pkg
```

Artifacts will be created in `dist-bin/` for macOS, Linux, and Windows.
